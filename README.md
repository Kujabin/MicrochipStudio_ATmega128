## 📌 프로젝트 설명

이 프로젝트는 **Microchip Studio** 기반으로 제작된 ATmega128용 테스트 코드입니다.  
UART 통신, 인터럽트 타이머 설정 등을 실습용으로 구성했습니다.

---

## 🧰 사용 기술

- Microchip Studio 7.0
- AVR ATmega128
- C Language
- Git, GitHub

## 🚀 실행 방법

1. Microchip Studio에서 `.atsln` 파일 열기
2. 빌드 후 ATmega128 보드에 업로드
3. UART 출력 확인 (115200 bps 등 설정에 맞게)
